# Code of Conduct

MicLayer follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be civil. Assume good faith.
- Critique code, not people.
- No harassment. No personal attacks. No politics or religion in issues — there are better places for that.
- English is the working language of issues and PRs, but typos and non-native phrasing are not a problem and will never be mocked.

## Scope

This applies to the repository, issues, pull requests, discussions, and any community space the project operates (Discord/Matrix if/when one exists).

## Enforcement

Report concerns by opening a private security advisory on the GitHub repository or by emailing the address listed on the maintainer's GitHub profile. Reports are handled confidentially. Maintainers may warn, mute, or ban contributors who breach this code.
